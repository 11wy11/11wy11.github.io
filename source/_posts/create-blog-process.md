###使用hexo+github创建个人博客并备份源码
###一、准备
1. 安装[node.js](https://nodejs.org/en/)
2. 安装[git-windows](https://gitforwindows.org/)
3. 安装hexo,hexo-cli
   ```
   $npm install hexo -g
   $npm install hexo-cli -g
   ```
2. 在GitHub上创建<username>.github.io的仓库
    ep. 11wy11.github.io
