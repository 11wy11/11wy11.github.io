# 常用转义字符
+ \\\ 反斜杠  
+ \\` 反引号  
+ \\* 星号  
+ \\_ 下划线  
+ \\{\\} 大括号  
+ \\[\\] 中括号  
+ \\(\\) 小括号  
+ \\# 井号  
+ \\+ 加号  
+ \\- 减号  
+ \\. 英文句号  
+ \\! 感叹号  
也可以用ASCII码代替
