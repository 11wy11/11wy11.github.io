###使用hexo+github创建个人博客并备份源码

###一、准备

  1. 安装node.js
  2. 安装git-windows

  3. 安装hexo-cli

  4. 在GitHub上创建.github.io的仓库

    e.g. 11wy11.github.io

  5. 配置SSH key
    打开本机的控制台，以windows系统为例，可以执行win+R 输入cmd打开
    执行

    然后连续按3次回车，最终会在用户目录下生成一个文件，找到目录.ssh/id_rsa.pub文件，用记事本打开全选复制
    在你的github上点击头像>设置进入设置页面，选择SSH and GPG keys, 点击New SSH key按钮，名称随意填写，将刚复制的内容粘贴到key,添加
    ！添加ssh key
    ###二、搭建流程

  6. 本地创建.github.io

  7. 在本地.github.io文件夹下初始化hexo项目并安装依赖

  8. 修改默认主题，可以在github上clone其他的主题，如
    yilia,
    icarus,
    indigo等

    • 以indigo为例，在本地.github.io文件夹下执行以下命令安装主题

    • 修改工程中的_config.yml配置文件，将themes: landscope改为themes: indigo

    • 依赖安装分别安装Less,Feed,Json-content依赖，执行如下命令

    • 执行

      deploy:
      type: git
      repo: git@github.com:/.github.io.git
      branch: master

      .deploy_git/
      node_modules/

      #git初始化
      git init
      #创建hexo分支，用来存放源码
      git checkout -b hexo
      #git 文件添加
      git add .
      #git 提交
      git commit -m “init”
      #添加远程仓库
      git remote add origin git@github.com:/.github.io.git
      #push到hexo分支
      git push origin hexo
      `
      执行hexo g -d生成网站并部署到GitHub上
      这样一来，在GitHub上的git@github.com:/.github.io.git仓库就有两个分支，一个hexo分支用来存放网站的原始文件，一个master分支用来存放生成的静态网页。

  9. 其他设备更新博客步骤：
    1、先安装hexo
    $ npm install -g hexo-cli
    2、存在github上的git clone下来
    git clone git@github.com:LJaer/ljaer.github.io
    3、项目文件夹下
    $ npm install
    $ npm install hexo-deployer-git
    4、重新配置github和coding的公钥
    5、编写博客push 到远程分支
